# Phishing URL Detection Website

## Overview

The Phishing URL Detection Website is a cybersecurity-based web application designed to identify whether a URL is legitimate or phishing. The project helps users stay protected from malicious websites and online scams by analyzing suspicious links.

## Features

* Detects phishing and safe URLs
* User-friendly web interface
* Real-time URL analysis
* Fast and accurate predictions
* Cybersecurity awareness project

## Technologies Used

* HTML
* CSS
* JavaScript
* Python
* Machine Learning

## How It Works

1. User enters a URL into the website.
2. The system analyzes the URL using predefined rules or a machine learning model.
3. The application predicts whether the URL is:

   * Safe
   * Phishing
